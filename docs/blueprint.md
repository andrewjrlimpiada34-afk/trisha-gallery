# **App Name**: Croquette Diary

## Core Features:

- Romantic Sensory Activation: A glassmorphism central button that triggers a synced experience of fading music, screen dimming, and pink fireworks.
- Cinematic Particle System: High-performance CSS/Canvas fireworks animation featuring soft pink, white, and gold particles with realistic fading effects.
- Polaroid Memory Carousel: An interactive slideshow featuring tilted, floating polaroids with parallax transitions and auto-zoom effects.
- AI Caption Refiner: An AI tool that can take a simple caption and transform it into a poetic or nostalgic diary entry in a specific romantic tone.
- Typewriter Narrative Effect: Captions and titles animate into view letter-by-letter as the user browses through different memory photos.
- Ambient Decorative Layer: Minimalist background animations including falling pink petals, subtle bokeh lights, and floating sparkles.
- Seamless Audio Orchestrator: Audio management system that handles the smooth fade-in and fade-out of the romantic score based on the modal state.

## Style Guidelines:

- Primary color: Soft Rose Pink (#EF94A7), a gentle and nostalgic hue that evokes warmth without being aggressive.
- Background: Whisper Pink (#FCF6F7), a nearly white, warm-toned grey-pink that provides ample whitespace for a clean layout.
- Accent color: Enchanted Lavender (#D584C8), providing an analogous contrast for buttons and key icons while maintaining a soft palette.
- Headline font: 'Playfair Display', a high-contrast serif for a sophisticated, high-end romantic feel. Body font: 'PT Sans' for warm, readable, and humane content.
- Fine-lined, minimalist stroke icons with soft glows, specifically focusing on the fireworks and music symbols to match Apple's clean design aesthetic.
- Apple-inspired minimalist single-page layout with extensive whitespace, utilizing glassmorphism modals with backdrop-blur for content overlays.
- Physics-based movement for the floating polaroids, subtle pulse animations for the CTA, and fluid page transitions between the landing state and the memory modal.
